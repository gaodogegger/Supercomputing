## IF - ELSE推导式

 `def abs1(n):`

​	`return n if(n>=0) else -n`

等价于

`def abs1(n):`

​	`if n>=0:`

​		`return n`

​	`else:`

​		`return -n`

可以使代码更简洁

## MATCH - CASE语句

Match后的对象会依次与case后的内容进行匹配，如果匹配成功，则执行匹配后的表达式，否则直接跳过，_可以匹配一切

EG:

`def abc(n):`

​	`match n:`

​		`case 1:`

​			`return"a"`

​		`case 2:`

​			`return"b"`

case匹配多个条件时，用|隔开
